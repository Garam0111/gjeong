# Bits of Codes

Repository of bits of codes that originally written or adapted by Garam Jeong. Purposes of adaptation was to apply the original codes for a certain target data sturcture. 

  - Original: Originally written by Garam Jeong
  - Adpated: Adapted from oroginal scripts. The source of original scripts and the major adaptation were marked at the head descirition of the script.

## Projects summary:
 - COCOA (2024-2025): A longitudinal study on semantic learning of alphabet letters using visual and auditory stimuli before and after formal schooling.
 - EMPRISE (2023-2025): A study to define topographic distribution of numerosity sensitive neuronal population base on a computational model in visual and auditory modality.  
   ref:  [High-field fMRI at 7 Tesla reveals topographic responses tuned to number in the developing human brain, Developmental Cognitive Neuroscience,2025](https://doi.org/10.1016/j.dcn.2025.101598),
     [Population Coding for Visual and Auditory Quantity in Human Numerotopic Maps, Communications Biology, 2026](https://doi.org/10.1038/s42003-026-09752-2)
 - One2Many (2020-2022): A dynamic functional connectivity model to characterize differences in default mode network configurations across internally generated mental
states like resting state and autobiographical memory retrieval state.  
   ref:  [Dynamic Adaptation of Default Mode Network in Resting state and Autobiographical Episodic Memory Retrieval State, BioArxiv, 2023](https://doi.org/10.1101/2022.08.30.505423)
